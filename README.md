# Landing Page Starbucks

Este é um projeto de landing page para a Starbucks, desenvolvido com HTML e CSS.

## Descrição

Uma página de destino simples e responsiva para apresentar produtos e informações da Starbucks.

## Estrutura do Projeto

- `index.html`: Página principal
- `src/css/style.css`: Estilos CSS
- `src/img/`: Imagens utilizadas

## Como Usar

1. Clone o repositório
2. Abra o arquivo `index.html` em um navegador web

## Tecnologias Utilizadas

- HTML5
- CSS3
